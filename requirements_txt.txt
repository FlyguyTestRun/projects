# ==============================================================================
# NOTES FOR STUDENTS
# ==============================================================================
#
# Installing Dependencies:
# 1. Install all: pip install -r requirements.txt
# 2. Install specific: pip install fastapi uvicorn
# 3. Upgrade all: pip install -r requirements.txt --upgrade
#
# Managing Dependencies:
# 1. Add new package: pip install <package>
# 2. Freeze current: pip freeze > requirements.txt
# 3. Remove package: pip uninstall <package>
#
# Version Pinning:
# - Exact version: package==1.2.3 (recommended for production)
# - Minimum version: package>=1.2.3
# - Compatible version: package~=1.2.3 (allows 1.2.x)
# - Range: package>=1.2,<2.0
#
# Development vs Production:
# - For production: Use exact versions (==)
# - For development: Can use minimum versions (>=)
# - Consider: requirements-dev.txt for dev-only packages
#
# Security:
# - Run: pip install safety
# - Check: safety check
# - Update vulnerable packages immediately
#
# Docker Optimization:
# - Copy requirements.txt before source code (better caching)
# - Use --no-cache-dir with pip install (smaller images)
# - Consider multi-stage builds
#
